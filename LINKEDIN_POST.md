# LinkedIn Post - Quantum Conduit Package Release

## Option 1: Professional & Technical

🚀 Excited to announce the release of **Quantum Conduit v0.0.2** on PyPI!

Quantum Conduit is the world's first PyTorch-native quantum statevector library designed specifically for quantum machine learning. Built from the ground up as a first-class PyTorch library (not a wrapper), it provides clean, low-level abstractions that integrate seamlessly with PyTorch's computational graph.

**Key Features:**
✨ Native autograd support throughout the quantum stack
📦 Full batch processing optimized for ML workloads
🔌 Zero-friction integration with PyTorch neural networks
⚛️ Complete quantum computing primitives: VQE, QAOA, adiabatic evolution, fermion-to-qubit mappings, and more

If you're building quantum ML models and already know PyTorch, Quantum Conduit provides the most natural integration—no plugin layers, no wrappers, just quantum computing primitives that work exactly like PyTorch tensors.

**Get started:**
```bash
pip install qconduit
```

🔗 Learn more: https://pypi.org/project/qconduit/
📚 Cite in research: https://doi.org/10.5281/zenodo.17599984
